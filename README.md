# project-dpwims
Questo progetto è sviluppato nell’ambito del corso B032427 (B335) – Distributed Programming for Web, IoT and Mobile Systems per l’anno accademico 2025–2026. L’obiettivo del progetto è progettare e realizzare un sistema distribuito che integri componenti web, IoT e/o mobile, applicando i principali concetti di programmazione distribuita.
